# dhinkachicka

Minimal Vite + React starter scaffold.

Getting started

1. Install dependencies

```bash
cd /Users/abhinav/work/dhinkachicka
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Build

```bash
npm run build
npm run preview
```

Notes

- This repo contains a minimal `src` with `App.jsx` and `main.jsx`.
- Uses Vite for fast development.
